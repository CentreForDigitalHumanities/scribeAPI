<h1> Scribe API </h1>
Scribe is an framework for generating crowd sources transcriptions of image based documents. 

<h2>Domain Overview</h2?
There are a number of domain entities in Scribe:

- Project
- Subject
- Workflow
- Task
- Classification
- User/Volunteer
